A - Этот селектор выбирает тип элементов, к которым будет применяться правило.

* - Выбирает все элементы.

.class - Выбирает элементы, основываясь на значении их class атрибута

#id - Выбирает элементы, основываясь на значении их id атрибута

A>B - Комбинатор '>' выбирает только те элементы, которые являются дочерними непосредственно по отношению к указанному элементу.

A B - Комбинатор ' ' (пробел) выбирает элементы, которые находятся внутри указанного элемента (вне зависимости от уровня вложенности).

A+B - Комбинатор '+' выбирает элемент, который находится непосредственно после указанного элемента, если у них общий родитель.

A~B - Комбинатор '~' выбирает элементы, которые находятся на этом же уровне вложенности, после указанного элемента, с тем же родителем.

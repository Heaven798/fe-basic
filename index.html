  • git status - Определения, какие файлы в каком состоянии находятся
  • git checkout - Для переключения на существующую ветку
  • git clone - Клонирование существующего репозитория
  • git push - Отправка изменений в удаленный репозиторий
  • git add - Для того чтобы начать отслеживать новый файл
  • git remote - Выведет названия доступных удалённых репозиториев
  • git commit - Для записи индексированных изменений в репозиторий
  • git branch - Показывает список имеющихся веток в репозитории
